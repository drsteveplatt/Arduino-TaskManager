void ring< T >::clear

Removes all elements from a ring.

Removes all elements from a ring. This will call the destructor of all stored objects. Calling on an empty ring will have no effect. The ring will be empty after this is called. Returns nothing.

◆ operator==()

template<class T >
bool ring< T >::operator== ( ring< T > &  r) const
inline

Compare two rings for equality.

Compares the two rings. This is a lightweight comparion. It compares the top level data (are they pointing to the same data components), but does not compare data components. As such, two rings separately constructed with the same data values will not be equal. But two rings constructed with the operator= will (provided neither have been modified).

Parameters
r- the ring that the current ring is being compared against.
