bool _TaskManagerTask::isRunnable ( )
protected

Determine whether or not a task can be run NOW.

Determine whether or not a task is runnable. A task can be run if and only if all of the following are true: (a) it is not suspended, (b) it is not waiting for a message, and (c) either it is not waiting for a time or that time has passed.
