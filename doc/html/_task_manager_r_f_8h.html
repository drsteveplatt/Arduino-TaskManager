#define TASKMGR_RADIO   TASKMGR_RADIO_RF24

Defines the class of radio transciever in use.

TaskManager has the capacity to support different radio transcievers. This defines which is in use.

Possible values are

  • 0 - No radio is in use, and the radio routines will not be compiled into the program.
  • 1 - The RF24 transciever is in use

The default value when TaskManagerRF.h is #include'd is TASKMGR_RADIO_RF24.
