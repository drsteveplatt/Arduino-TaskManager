#define TASKMGR_MESSAGE_SIZE   (TASKMGR_MAX_PAYLOAD-1-sizeof(tm_nodeId_t)-2*sizeof(tm_taskId_t))

The maximum size of a message passed between tasks

This defines the maximum size for an inter-task message. It is constrained, in part, by plans for RFI communication between tasks running on different devices.

Note that the NRF24's max payload size is 32. Message overhead is 3 bytes for cmd, src node, src task and 1 byte for target task.

ESP-NOW max payload is 250, overhead is 4 for cmd/src and 1 for target. Message overhead is 4 bytes for cmd, src node, src task, and 1 byte for target task. THIS MAY CHANGE IF WE ALLOW MORE THAN 256 TASKS
