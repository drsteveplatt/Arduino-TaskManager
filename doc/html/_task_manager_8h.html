#define TASKMGR_MESSAGE_SIZE   (32-3)

The maximum size of a message passed between tasks

This defines the maximum size for an inter-task message. It is constrained, in part, by plans for RFI communication between tasks running on different devices.

Note that the NRF24's max payload size is 32. Message overhead is 3 bytes.
